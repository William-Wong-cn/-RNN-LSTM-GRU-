# 基于RNN/LSTM/GRU的人名国籍分类系统

**技术栈**：Python · PyTorch · RNN / LSTM / GRU · Matplotlib · 模块化工程  
**项目时长**：2024.10 – 2024.12（独立完成）

### 项目简介
基于经典“根据人名预测国籍”任务，使用 PyTorch 从零实现并系统对比 RNN、LSTM、GRU 三种循环神经网络，构建完整、可复现、可视化的深度学习工程项目。

### 核心亮点
- 使用 PyTorch 从零实现 RNN、LSTM、GRU 三种经典循环神经网络（字符级 One-Hot 输入）
- 在 18 个国家、20,074 条人名数据集上完成国籍分类，**Top-1 准确率最高达 73.8%（GRU 最佳）**
- 完整工程化设计（6 大模块）：
  - `config.py`：统一超参数管理  
  - `data_utils.py`：自定义 Dataset 与 DataLoader  
  - `models.py`：三种网络清晰实现  
  - `train.py`：训练 + 每 100 步记录指标 + 自动保存模型  
  - `evaluate.py`：自动绘制 Loss / Accuracy / 训练耗时三图对比  
  - `main.py`：一键训练、绘图、预测
- 训练结束自动生成三图对比（Loss、Accuracy、训练耗时）并保存为 `comparison.png`
- 支持实时预测任意姓名，返回 Top-3 国籍概率  
  示例：  
  `Suzuki` → Japanese 99.3%  
  `Ivanov` → Russian 98.7%  
  `Zhang`   → Chinese 99.1%

### 实验结果
| 模型  | Top-1 准确率 | 3 epoch 训练耗时 |
|-------|--------------|------------------|
| RNN   | 68.4%        | 182s             |
| LSTM  | 72.1%        | 198s             |
| GRU   | **73.8%**    | **165s**         |
